## Tintsabers

Tintsabers is a cosmetic datapack that provides toggleable tintsabers: swords that have a light-like blade that changes color based on team color or sword trim.

Re-implemented from an old 1.13+ version data-&-resource pack written for an SMP, Tintsabers aims to be vanilla-friendly as its primary purpose is cosmetic changes.

**Features**
- Convert any vanilla sword into a tintsaber, retaining their base attack stats
- Tintsabers will automatically change color based on your minecraft team, if a color has been assigned
- Tintsaber team-shading can be overrode by using armor trim materials on them
  - This change is permanent on a tintsaber, choose wisely
- Toggleable blade extension and retraction by crouching with the saber in your main hand

**Future Planned Features**
- Updates as needed; subtle particle & sound effects later


## Technical Info
This pack utilizes custom trim definitions and as such gets marked as experimental when loaded in a world. Make backups accordingly if you have concerns regarding this.

**IMPORTANT:** Please remember to use the included resourcepack listed in the release's additional files to actually see the custom models.
